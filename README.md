#Campus Energy-Use Dashboard
This project analyzes electricity consumption across multiple campus buildings and produces a complete end-to-end data pipeline including ingestion, aggregation, visualization, and summary reporting.

#Objectives
* Load and validate multiple monthly energy CSV files.
* Clean, merge, and organize time-series electricity-usage data.
* Compute daily & weekly aggregates and building-wise summaries.
* Use OOP classes (Building, MeterReading, Manager) to model real data
* Generate a multi-chart dashboard with Matplotlib.
* Export cleaned data, summary tables, and an executive text report.

  **Tools Used**

      Python
      Pandas
     Matplotlib
    OOP principles
    File handling

**Results**
Clean combined dataset
Daily & weekly consumption graphs
Building-wise summary table
Executive summary report
